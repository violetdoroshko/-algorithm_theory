Задача  31

 


Условие

 

Найти вершины, через которые проходит нечетное число путей максимальной длины,  и удалить (правым удалением) ту из них, ключ которой наибольший. Выполнить прямой (левый) обход полученного дерева.

 

Входные данные 

 

tst.in содержит последовательность чисел - ключей дерева.

 

Выходные данные 

 

tst.out содержит массив вершин, полученный прямым левым обходом итогового дерева.

 

Пример входных данных 

 

10

1

12
 

Пример выходных данных

 

10

1
