Задача. «Открытки и конверты».


Условие

 

Имеется N прямоугольных конвертов и N прямоугольных открыток различных размеров. Необходимо определить, можно ли разложить все открытки по конвертам так, чтобы в каждом конверте было по одной открытке. Открытки нельзя складывать, сгибать и т.п., но можно помещать в конверт под углом. Например, открытка с размерами сторон 5:1 помещается в конверты с размерами 5:1, 6:3, 4.3:4.3, но не входит в конверты с размерами 4:1, 10:0.5, 4.2:4.2.

 

Входные данные

    Входные данные находятся в файле input.in.

·                   Первая срока содержит число N - количество открыток (конвертов).

·                   Затем идут N строк файла. Каждая из строк содержит два числа, которые задают размеры очередной открытки (в строке числа разделены пробелами).

·                   Затем идут N строк файла. Каждая из строк содержит по два числа, которые задают размеры очередного конверта (в строке числа разделены пробелами).

 

 

Выходные данные 

 

Выходные данные находятся в файле output.out.

Если все открытки можно разложить, то единственная строка файла содержит сообщение  YES .

Если все открытки нельзя  разложить, то первая строка выходного файла содержит сообщение NO,

 а вторая строка содержит максимальное число открыток, которые можно разложить по конвертам.

 

Пример входных данных

2

7      1.4

6.5   5

7      7

6      6

 

Пример выходных данных 

 
YES
