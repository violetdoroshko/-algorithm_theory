Задача 6. «Строго возрастающая без разрывов подпоследовательность».   

Время на тест 1 секунда

 


Условие

 

Необходимо из заданной числовой последовательности  A, состоящей  из n элементов, вычеркнуть  минимальное количество элементов так,  чтобы оставшиеся  элементы образовали строго возрастающую  подпоследовательность элементов. Построенный алгоритм должен иметь трудоемкость O(n log n).

  

 

 

Входные данные

 

Входные данные подготовлены в файле input.txt:

1.                 первая строка файла содержит число   n (1 < n ≤ 100000);

2.                 следующая строка содержит n элементов последовательности A, которые разделены пробелом(элементы последовательности целые натуральные числа не превосходящие 1000000000).

 

Выходные данные

 

Выходные данные  заносятся  в файл output.txt : len - длина строго возрастающей  подпоследовательность элементов.

 

 

Пример входных данных

 

6

1 2 3 4 7 6

 

 

 

Пример выходных данных

 

5
